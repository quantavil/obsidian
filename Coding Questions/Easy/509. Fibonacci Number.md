## Approach 1 : Memoization  - Top Down Approach DP
```python
class Solution:
    cache = {}
    
    def fib(self, n: int) -> int:
        if n in self.cache: # check if the result has already been cached
            return self.cache[n]

        # calculate the result recursively
        if n <= 1:
            self.cache[n] = n
        else:
            self.cache[n] = self.fib(n-1) + self.fib(n-2)
        
        return self.cache[n]

```
